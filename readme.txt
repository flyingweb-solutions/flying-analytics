== Description ==

Load Optimized & Self-hosted Google Analytics

## Features
- Self-host Google Analytics
- Super minified JS files
- Choose between 4 different JS methods
- Disable tracking for logged in admins, editors & contributors
- Non-render blocking!

== Installation ==

1. Visit 'Plugins > Add New'
1. Search for 'Flying Analytics'
1. Activate 'Flying Analytics' from your Plugins page.
1. Visit 'Settings -> Flying Analytics' to configure
